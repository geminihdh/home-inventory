# 내 손안의 우리집 물건 관리 (Native App) - 디자인 명세서

이 문서는 React Native(Expo)를 사용하여 개발되는 개인용 물건 관리 네이티브 앱의 설계를 정의합니다. 이 앱은 휴대폰 파일 시스템에 직접 데이터를 저장하여 데이터 안전성을 극대화합니다.

## 1. 개요
- **목적:** 집안의 물건을 카테고리별로 등록하여 위치, 유통기한 등을 관리.
- **주요 가치:** 
  - **데이터 보존:** 휴대폰 내부 파일 시스템에 직접 저장하여 앱 삭제 외에는 데이터 유실 방지.
  - **네이티브 기능:** 홈 화면 위젯 및 푸시 알림을 통해 능동적인 물건 관리.
  - **오프라인 우선:** 인터넷 연결과 무관하게 즉시 작동.

## 2. 기술 스택
- **Framework:** React Native with Expo
- **Data Storage:** `expo-file-system` (파일 기반 데이터 저장)
- **Native Features:**
  - `expo-notifications` (유통기한 알림)
  - `expo-widgets` (홈 화면 위젯)
- **Data Format:** JSON (직렬화하여 파일로 저장)

## 3. 데이터 모델
`InventoryItem` 인터페이스:
- (PWA 버전과 동일한 필드 + 알림 상태 필드 추가)
- `notificationId`: string (푸시 알림 식별자)

## 4. 주요 기능
- **파일 기반 데이터 관리:** 데이터를 JSON 파일로 저장/읽기.
- **푸시 알림:** 유통기한 만료 전/당일 알림 기능.
- **홈 화면 위젯:** 만료 임박 물건 목록 표시 위젯.

## 5. 보안 및 데이터 정책
- 사용자의 모든 데이터는 앱 전용 폴더에 저장되며 외부로 전송되지 않음.
- 로그인 없음.
